# AutoClipr social media brand assets

## Profile picture (all platforms)

YouTube, Instagram, Twitter/X, and Facebook crop profile photos to a **circle**.  
Use the **circle-safe** file (icon only, centered, no text).

| File | Use |
|------|-----|
| **`social-profile.png`** | **Upload this** — fits circular crop |
| `social-profile-circle.png` | Same as above |
| `social-profile-circle.svg` | Editable source (icon only) |
| `social-profile.svg` | With text — **not** for profile pics (gets cut off) |

**Recommended size:** 1080×1080 px (minimum 400×400)

**Do not use** the version with "AutoClipr" text below the icon — it is cropped on Instagram/YouTube.

## YouTube

| File | Size | Use |
|------|------|-----|
| `youtube-banner.png` | 2560×1440 | Channel banner |
| `youtube-banner.svg` | Scalable | Edit source |

## Twitter / X & Facebook

| File | Size | Use |
